- **Regla de datos**: todo contenido no confirmado (partners, proyectos, reseñas, posts) lleva chip `DEMO` o placeholders neutros. No se maquetan clientes reales sin permiso.
- Naming público: `Proyectos` (no `Portafolios`); sección de landing con logos/empresas/trabajos = `Colaboraciones`.
- El hero replica la estructura del hero real protegido; en código no se rediseña.
